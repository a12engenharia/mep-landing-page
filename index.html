O que é o MEP

❌ Não é um curso de marketing.

❌ Não é um curso sobre Instagram.

❌ Não é uma coleção de aulas soltas.

É o sistema completo para operar um escritório de engenharia ou arquitetura lucrativo: mentalidade, captação, estrutura, processos e operação — com o Instagram entrando só como ferramenta.

Para quem é o MEP

Feito para quem está construindo, não para quem já chegou lá.

No Instagram, eu mostro o dia a dia real do meu escritório — decisões, dificuldades, acertos e erros. O MEP nasceu dessa mesma jornada: é para quem está começando a construir o próprio caminho, não para quem já tem um escritório grande e consolidado.
